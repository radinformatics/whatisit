What is it?

WhatIsIt is an annotation tool for experts to label radiology reports. It is an effort toward collaborative annotation to produce publicly available datasets for research. WhatIsIt will (eventually) allow for a combination of human- and algorithm- derived annotations of reports. This work is being done by the Langlotzlab at Stanford University.
